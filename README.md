# Light Maintenance - Official Website

This is the static landing page website for the **Light Maintenance** WordPress plugin.

## 🚀 Stack
- **HTML5**: Semantic and SEO optimized.
- **Tailwind CSS**: Loaded via CDN for zero-build configuration.
- **Vercel**: Optimized for Vercel's Edge Network.

## 📂 Site Structure
- `index.html`: Main landing page with features and download CTA.
- `changelog.html`: Version history timeline.
- `privacy.html`: Privacy policy (GDPR/Tracking info).
- `terms.html`: Terms of Service and Licensing (GPLv2).

## 🛠 Deployment
This site is designed to be deployed on **Vercel**.
1. Import this repository to Vercel.
2. Leave "Build Command" and "Output Directory" empty.
3. Deploy.
