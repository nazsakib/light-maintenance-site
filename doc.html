Installation

01 Via WordPress Dashboard

  1. Log in to your WordPress WP Admin area.
  2. Go to Plugins > Add New.
  3. Search for Sakibsnaz Light Maintenance Mode.
  4. Click Install Now and then Activate.

02 Manual Upload (FTP/Local)

Perfect for local development or manual server deployment. You can acquire the plugin files in two ways:

Via Official Website

  1. Visit lightmaintenance.site.
  2. Click the Get Plugin button in the top navigation.
  3. Save the .zip file to your computer.

Via WordPress.org

  1. Go to the Plugin Repository.
  2. Click the blue Download button.
  3. Save the .zip file to your computer.

Finalizing Installation

  1. Extract the downloaded sakibsnaz-light-maintenance-mode.zip folder.
  2. Use an FTP client or File Manager to navigate to your site's /wp-content/plugins/ directory.
  3. Upload the unzipped folder into that directory.
  4. Go to WP Admin > Plugins and click Activate on Light Maintenance.

03 Via WP-CLI

# Install and activate instantly

$ wp plugin install sakibsnaz-light-maintenance-mode --activate

Activation

Setting up is designed to be a one-click process. No onboarding wizards or popups.

  1. Navigate to Settings → Maintenance Mode.
  2. Toggle the Enable Maintenance Mode checkbox.
  3. Click Save Changes.

Note: Logged-in administrators bypass the splash screen automatically. To see the maintenance page yourself, log out or view your site in an Incognito/Private window.

Technical Details (SEO Safe)

For developers concerned about search rankings, Light Maintenance is the safest choice.

503

Native 503 Headers

We automatically serve a 503 Service Unavailable header. This prevents Google from de-indexing your site during maintenance.

JS

No Scripts Loaded

The maintenance screen does not load external CSS, Google Fonts, or JavaScript libraries. It is pure, fast HTML.
